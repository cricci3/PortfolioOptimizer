# Portfolio Optimizer API 📈
A lightweight, real-time API that transforms a list of tickers into an optimized investment portfolio using Modern Portfolio Theory (MPT).

## 🚀 Overview
The Portfolio Optimizer API allows users to input a collection of assets (Stocks, Crypto, ETFs) and receive the mathematically "best" weights for those assets. It handles the heavy lifting of data retrieval, statistical analysis, and quadratic programming optimization.
